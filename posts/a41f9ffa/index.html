关键问答

当前视频不是 RGB，为什么？

为了教学简单，本课使用灰度移动方块视频。真实视频通常是 RGB 或 VAE latent。

时空 patch 和图片 patch 的区别是什么？

图片 patch 只切空间。时空 patch 同时切时间和空间。

Sora 的 spacetime patches 是不是类似？

基本思想类似：把视频表示为时空 patch 序列。但工业级系统还有复杂的压缩、位置编码、变长处理和大规模训练。

一分钟复述

视频张量写作 [B,C,T,H,W]，比图片多时间维。Video VAE 将大尺寸 RGB 视频压缩到更小的 latent，降低计算量。Video DiT 会把 latent 视频切成时空 patch，每个 patch 变成 token。token 数由时间、空间尺寸和 patch size 决定。由于 attention 成本随 token 数平方增长，视频生成比图片生成更吃显存和算力。
