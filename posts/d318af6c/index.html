常见误区

一分钟复述

Image DiT 将图片切成 patch token，加入位置编码后送入 Transformer。时间 t 通过 adaLN 注入每个 DiTBlock。Transformer 输出 token 后，final projection 将每个 token 变成 patch 像素，depatchify 拼回图片形状。本课使用 Flow Matching，模型预测速度场，推理时通过 Euler ODE 从随机噪声更新到生成图片。
